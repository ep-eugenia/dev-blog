Javascript Objects


Objects are the foundations of JavaScript.

Anything that is not a primitive value is an object.

Objects are values and they serve the same purpose as primitive values : Representing information or data.


In JavaScript, an object can be thought of as a distinct entity that can have properties associated with it. These properties define the object’s characteristics and behavior.

The properties are also known as key-value pairs.


We can create an object using “literal” notation as shown below:


const obj = { };


EXAMPLE:

Here we create the oject 'person' and can also specify properties to be inserted at the time of the object's creation:


  • const person = {
  • firstName: 'John',
  • lastName: 'Müller',
  • dateOfBirth: '14th May',
  • friends: [ 'Tim', 'Alice' ]
  • };


The property names are treated as strings : this means firstName and 'firstName' are the same thing.

An object property name can be any valid JavaScript string, or anything that can be converted to a string, including the empty string.

The values can be any value type. That means, both primitive types and an object can be used as values.
