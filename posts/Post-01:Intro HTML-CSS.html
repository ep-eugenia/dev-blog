HTML and CSS

Welcome to the world of HTML


HTML is an acronym which stands for Hyper Text Markup Language and is the fundamental technology used to define the structure of a webpage.


Hypertext refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web.


HTML uses "markup" to annotate text, images, and other content for display in a Web browser.


An HTML document is a plaintext document structured with elements. Elements are surrounded by matching opening and closing tags. Each tag begins and ends with angle brackets (<>).
